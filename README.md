# Focusflow

A beautiful Pomodoro timer + todo list app with Apple-inspired design, 6 vibrant themes, and 8 synthesized notification ringtones.

**Live:** [pomodoro-todo-five.vercel.app](https://pomodoro-todo-five.vercel.app)

## Features

- **Pomodoro Timer** — Focus / Short Break / Long Break with animated ring progress
- **Custom Durations** — Set any duration (1–120 min) for each mode via Settings
- **Todo List** — Add, complete, delete tasks; filter by All / Active / Completed
- **Task Focus** — Click a task to pin it; pomodoro count auto-increments on the active task
- **6 Themes** — Sunset, Ocean, Lavender, Midnight, Forest, Sakura — all with glassmorphism
- **8 Ringtones** — Classic Chime, Digital Beep, Happy Melody, Gentle Wind, Urgent Alert, Piano Keys, Bird Song, Bubble Pop (Web Audio API, no external files)
- **Volume Control** — Adjustable 0–100% with persistent preference
- **Data Persistence** — Todos, theme, ringtone, volume, and custom times saved to localStorage
- **Keyboard Shortcuts** — Space (start/pause), R (reset), Esc (close settings)
- **Responsive** — Works on mobile and desktop

## Usage

Open `index.html` in any modern browser. No build step, no dependencies.
